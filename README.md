# Pokédex - Linha do Dragonite

Uma pokédex simples e interativa focada na linha evolutiva do Dragonite.

## Características

- **Quatro Pokémon**: Dratini → Dragonair → Dragonite → Mega Dragonite
- **Cards Informativos**: Altura, peso, tipos, habilidades e descrições
- **Design Responsivo**: Funciona em desktop, tablet e celular
- **Interatividade**: Clique nos cards para ver detalhes no console
- **Imagens Dinâmicas**: Sprites direto da PokéAPI

## Estrutura do Projeto
